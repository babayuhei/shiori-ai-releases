# Shiori AI Desktop — Releases

This repository hosts the **signed compiled artifacts** for Shiori AI Desktop.

- Source code: private at [babayuhei/shiori-ai](https://github.com/babayuhei/shiori-ai)
- Downloads: see the [Releases](https://github.com/babayuhei/shiori-ai-releases/releases) tab
- Auto-update: installed apps check `latest.json` on launch and update silently in the background

All uploads are automated from the source repo's `desktop-release.yml` workflow.
